# PDF Export Feature Guide

## Overview
Professional PDF report generation with comprehensive analytics data, charts, and insights.

## Features

### What's Included in the PDF

1. **Header Section**
   - Shop name
   - Report generation date
   - Selected time period
   - Professional branding

2. **Executive Summary**
   - Total revenue (quarter)
   - Total orders (quarter)
   - Total customers
   - Average order value
   - Completion rate

3. **Growth Metrics**
   - Month-over-month growth (orders & revenue)
   - Week-over-week growth (orders & revenue)
   - Percentage changes with +/- indicators

4. **Monthly Revenue Trend**
   - Last 6-12 months data
   - Order count per month
   - Revenue per month
   - Average order value

5. **Top Customers**
   - Top 10 customers by revenue
   - Total orders per customer
   - Total revenue per customer
   - Average order value

6. **Daily Performance**
   - Performance by day of week
   - Orders and revenue per day
   - Average order value per day

7. **Peak Hours Analysis**
   - Hourly performance data
   - Active hours only
   - Order count and revenue by hour

8. **Footer**
   - Page numbers
   - "Generated by Zaprint Analytics"
   - Confidential marking

## How to Use

### From Analytics Page

1. Navigate to Analytics page
2. Select desired time period (7d, 30d, or 90d)
3. Click "Export Report" button
4. PDF will automatically download

### File Naming

Format: `ShopName_Analytics_YYYY-MM-DD.pdf`

Example: `Soham_Shop_Analytics_2024-03-15.pdf`

### Download Location

PDF saves to your browser's default download folder.

## Troubleshooting

### PDF Not Downloading

**Problem:** Click export but nothing happens

**Solutions:**
1. Check browser popup blocker
2. Allow downloads from your site
3. Try different browser
4. Check browser console for errors

### Missing Data in PDF

**Problem:** Some sections show "No data available"

**Solutions:**
1. Ensure orders exist in database
2. Check selected time period
3. Verify analytics views are working
4. Run database queries to confirm data

### PDF Looks Broken

**Problem:** Tables overflow or text is cut off

**Solutions:**
1. Clear browser cache
2. Update to latest version
3. Check if data is too long
4. Report issue with sample data

### Large File Size

**Problem:** PDF file is very large

**Solutions:**
1. Reduce time period
2. Limit customer list
3. Remove unnecessary data
4. Compress PDF after generation
